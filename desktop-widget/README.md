# Планер Widget

Малко Electron приложение, което показва днешните задачи в постоянен, always-on-top
прозорец на десктопа — обвивка около `frontend/widget.html` (същите api.js/stores.js
модули като главното приложение, само компактен изглед).

## Локално пускане

```bash
cd desktop-widget
npm install
npm start
```

Изисква бекендът да работи на `http://localhost:3000` (виж root README). Ако сървърът
е другаде, промени `serverUrl` в конфигурацията (десен клик върху tray иконата →
"Отвори конфигурацията").

## Инсталатор (.exe)

```bash
npm run dist
```

Създава инсталатор в `dist/` — след инсталация widget-ът тръгва автоматично с Windows
(може да се спре от tray менюто) и слага икона на десктопа.

## Управление

Widget-ът няма рамка/лента с бутони — влачи се за фон. Затварянето (× в горния десен
ъгъл, на логин екрана и след логване) напълно приключва приложението. Достъпва се и
през иконата в system tray (долу вдясно до часовника):

- **Покажи/Скрий** — превключва видимостта без да затваря приложението
- **Презареди** — презарежда изгледа (полезно след смяна на конфигурацията)
- **Стартирай с Windows** — включва/изключва автостарта
- **Отвори конфигурацията** — отваря папката с `config.json` (сървър адрес, размер/позиция на прозореца)
- **Изход** — същото като × бутона в самия widget
